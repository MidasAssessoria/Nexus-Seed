# Session diary

Append-only log written by `/reflect` at the end of each session.
Records facts (what was done) + reflection (Regret / Gratitude / Evolve) + next step.

Format:

```
## Session — YYYY-MM-DD (short description)

**Facts**
- files created/edited
- skills invoked
- decisions made

**Reflection**
- Regret: what you wouldn't repeat
- Gratitude: what worked and is worth keeping
- Evolve: what changes in the next session

**Next session**
- open items
- clear next step
```
